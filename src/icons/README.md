# Icon Placeholder Files

This extension requires icon files in PNG format. You can create them in the following ways:

## Option 1: Use the SVG template (recommended)
1. Open `icon.svg` in a graphics editor like Inkscape, GIMP, or an online tool
2. Export as PNG at these sizes: 16x16, 32x32, 48x48, and 128x128 pixels
3. Save them as `icon16.png`, `icon32.png`, `icon48.png`, and `icon128.png` in this folder

## Option 2: Use an online converter
1. Go to https://www.adobe.com/express/feature/image/resize or similar
2. Upload `icon.svg`
3. Resize to 128x128, 48x48, 32x32, and 16x16
4. Save each as the appropriate filename

## Option 3: Create simple placeholder PNGs
You can use any image editor to create simple colored squares as temporary icons:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

The SVG icon shows a circular arrow (refresh/reload symbol) with a download arrow, representing the clear cache and reload action.
